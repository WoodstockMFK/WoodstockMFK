# Woodstock Modelflyveklub

Hjemmeside for Woodstock Modelflyveklub.

Siden er lavet for at give nye og nysgerrige besøgende et indtryk af klubben, flyvepladsen, fællesskabet og glæden ved modelflyvning.

## Om siden

Her kan man læse om:

- hvorfor modelflyvning er en spændende hobby
- hvilke typer modeller vi flyver med
- hvordan nye piloter kan komme i gang
- hvor klubben holder til
- hvornår vi flyver og mødes
- lidt om klubben og fællesskabet
- billeder fra flyvepladsen og klublivet

## Formål

Målet med hjemmesiden er at gøre det nemt at finde praktisk information om klubben og samtidig vise, at Woodstock Modelflyveklub er et sted med plads til både begyndere og erfarne entusiaster.

Siden skal være enkel, hurtig og let at gå til — både for besøgende og for dem, der vedligeholder den.

## Teknisk

Hjemmesiden er bygget i:

- HTML
- CSS
- JavaScript
- INGEN COOKIES!!

Projektet er bevidst holdt enkelt uden frameworks, så det er nemt at arbejde videre med.

## Indhold

Projektet består blandt andet af:

- `index.html` – forside
- `om-os.html` – side om klubben
- `style.css` – layout og design
- `images/` – billeder til siden og fotogalleri
